Me-Drawing

Team Members: Q (Solo Project)

Application Overview

Me-Drawing is a small application that takes a user’s uploaded photo and converts it into a line-art drawing using Python Turtle graphics. The program extracts the outlines of the image and draws them using turtle to create a sketched version of the person.

Technology Overview

  • Languages: Python, HTML, CSS
  • Libraries/Frameworks: Turtle, OpenCV (image edges), PIL
  • Platform: Desktop application / Python environment
  • Hosting: Local machine (code runs in Python)

Proposal Last Updated

November 19, 2025

Feature List (Prioritized)

  1. Upload an image of the user
  2. Convert the image into contour/outline data
  3. Generate a turtle drawing based on the outlines

Advanced Technical Features
