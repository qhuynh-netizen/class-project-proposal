Me-Drawing (Python Turtle Cartoon)

Team Members: Q

Application Overview

The Me-Drawing project uses Python’s Turtle graphics to draw a cartoon version of a user’s uploaded image. The program converts a real picture into simplified outlines and then recreates it as a cartoon-like sketch using turtle drawing commands.

Technology Overview

  • Languages: Python
  • Libraries/Modules: Turtle, OpenCV (for line detection), JSON handling
  • Platform: Desktop Python program
  • File Input: User uploaded image (PNG/JPG)

Proposal Last Updated

November 2025

Feature List (Prioritized)

  1. Upload an image and convert it into cartoon outlines
  2. Generate turtle paths from image contours
  3. Python Turtle draws the cartoon version of the user

Future Feature List

  • Add color-filling to cartoon shapes
  • Allow different cartoon styles (simple, detailed, anime)
  • Export turtle drawing as an image file

Basic Technical Features

  • Loading and processing user images
  • Edge detection using OpenCV
  • JSON file storing simplified drawing paths

Advanced Technical Features

  • Algorithmic contour simplification for cartoon effect
  • Automatic path smoothing and coordinate scaling
  • Real-time turtle rendering of complex image outlines
